= How can i get support if my Lead form builder is not Working ? =

If you're facing any problem while using our Lead form builder plugin, Please first follow the below steps :
1. Deactivate all plugins you are using and check if the problem is still occurring with our Lead form builder .
2. Check the theme and plugin version, if it is latest or not.
3. Check the plugin with default theme, to make sure error is not due to theme you're using.
